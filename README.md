# 🇻🇳 Vietnam Fun Challenge

An elegant, interactive quiz about Vietnamese culture, food, and modern society.

## ✨ Features

- **Beautiful Vietnamese-inspired design** - Traditional red/gold colors with lotus and bamboo motifs
- **10 engaging questions** - Food Frenzy challenge, emoji decoders, cultural facts
- **Offline-ready** - No internet required, single HTML file
- **Auto-scoring** - Instant feedback with prize tier recommendations
- **Mobile-friendly** - Responsive design works on all devices

## 🎮 Live Demo

Visit: **[https://ngankp.github.io/vietnam-quiz/](https://ngankp.github.io/vietnam-quiz/)**

## 🎯 Content

1. **Food Frenzy** - 30-second timed challenge to name Vietnamese dishes
2. **Emoji Decoders** - Guess iconic foods from emojis (🍜🐄🌿 = Phở!)
3. **Surprising Facts** - True/False about Vietnamese culture
4. **Modern Vietnam** - Social media, demographics, and contemporary culture
5. **History & Geography** - S-shaped country, coffee culture, French influences

## 🏆 Quiz Structure

- **10 questions total**
- **3-5 minutes** to complete
- **Prize tiers:**
  - 🏆 80%+ correct: LEGENDARY!
  - ⭐ 60-79% correct: Amazing!
  - 😊 40-59% correct: Great job!
  - 🎈 Below 40%: Nice try!

## 🎨 Design
